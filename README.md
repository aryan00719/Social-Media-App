# Social-Media-App

Full-stack social media application built with React and Node.js featuring real-time interactions, REST API integration, and dynamic state management.

## Project Setup

This project was bootstrapped with React + Vite.

### Prerequisites
- Node.js
- npm

### Installation
1. Clone the repository
2. Install dependencies for the frontend and backend:
   ```bash
   npm install
   cd backend && npm install
   ```

### Running the App
- Start the backend: `node backend/server.js`
- Start the frontend: `npm run dev`

### Features
- Minimal setup to get React working in Vite with HMR.
- ESLint rules for code quality.
